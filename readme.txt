YD Culqi payment gateway for AliDropship provides an easy way to take credit card payments on your online store using Culqi.

== Description ==
YD Culqi payment gateway for AliDropship is a quick and easy solution for taking credit card payments on your online store using Culqi. To register and set up your Culqi account, visit [this page](https://culqi.com/).
Culqi is fully compatible with WordPress and AliDropship latest versions.
If you find something wrong in the plugin's functioning, I am always open for discussion and glad to hear any suggestions, questions, and notes.

**Features:**
- Simple setup
- A popup window on your checkout page
- Compatible with AliDropship

= Support =
In case you have any questions or need technical assistance, please, contact our support team - support@alidropship.com

== Installation ==
To install the plugin, please follow this instruction:
1. Upload the `ds-culqi` folder to the `/wp-content/plugins/` directory, or install the extension through the WordPress plugins screen directly.
2. Activate the extension through the 'Plugins' menu in WordPress.
3. The settings will be available in AliDropship - Settings - Payments.

== Frequently Asked Questions ==
= How to setup a Culqi account? =
To register and set up your Culqi account, visit [this page](https://culqi.com/).
= How can I find this option in my WP admin area?
Open AliDropship - Settings - Payments, select Culqi in Credit/Debit Card Section, and fill in all settings fields.
